How to use FlashCards

1 Create a deck manually or use AI Generate to build cards instantly
2 Import cards from a CSV file using the 📊 Import CSV button
3 Choose a study mode: Flashcard, Multiple Choice, or Type Answer
4 Rate cards as Missed / Vague / Got it — the app schedules smarter reviews
5 Use the 🍅 Pomodoro timer to stay focused during sessions
6 Check 🏆 Awards to track your achievements and milestones
7 Keyboard: Space flip · ← Missed · → Got it
